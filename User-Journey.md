A user in Microsoft 365 is created by an administrator in the Microsoft 365 Admin Center.
The admin signs in with their admin account.
They go to the “Users” section and select “Active users.”
Then they click “Add a user” to start creating a new account.
The admin enters the user’s name and display name.
A unique username and domain are assigned.
A password is generated automatically or set manually.
The admin can assign roles such as user or administrator.
After entering all details, they save the new user.
The account is created in Microsoft Entra ID and is ready for sign-in.

A license can be assigned during the creation process or afterward.
To assign it later, the admin selects the user in the admin center.
They open the “Licenses and apps” section.
The admin chooses an available subscription plan.
The license is then assigned to the user account.

Assigning a license enables access to Microsoft 365 services like email and collaboration tools.
It also determines features such as storage, security, and compliance capabilities.
Without a license, the user cannot fully use Microsoft 365 services.
