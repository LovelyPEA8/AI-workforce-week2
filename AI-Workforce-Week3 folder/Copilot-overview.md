Microsoft Copilot is an AI assistant integrated into tools like Microsoft 365 and Windows 11 to help users draft content, summarize information, analyze data, and automate tasks.
It combines large language models (including technology from OpenAI) with organizational context to provide relevant, real-time assistance.

Copilot depends on identity, data, and permissions to function correctly.
Identity is managed through Microsoft Entra ID, which verifies who the user is.
Data comes from sources like OneDrive, SharePoint, and emails or chats the user can access.
Permissions ensure Copilot only retrieves and presents information the user is authorized to see.

Copilot is not an independent system with full access to all organizational data.
It does not override security controls or permissions.
It is not a replacement for human oversight, decision-making, or data governance.
